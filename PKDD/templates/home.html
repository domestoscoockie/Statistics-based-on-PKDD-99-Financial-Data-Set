Statistics Based on Data from the 'PKDD'99 Discovery Challenge Financial Dataset

Made by Karol Grabowski

The following data represent a statistical dataset derived from financial data used in the 'PKDD'99 Discovery Challenge. The dataset includes information on transactions, credits, and other financial operations useful for further analysis. The data has been cleaned and transformed by me into a format suitable for a PostgreSQL database. I have also prepared charts and statistics using Dash based on this dataset.
